## Описание задания

В рамках лабораторной работы реализованы два набора задач по работе с реляционными базами данных через Python (модуль `sqlite3`):

- **Задачи №1** — 5 программ, выполняющих SELECT-запросы к базам данных `films.db` и `music.db`
- **Задачи №2** — 10 задач: 8 программ с функцией `get_result(name)` для модификации данных (INSERT/UPDATE/DELETE), а также системы регистрации и авторизации пользователей

Базы данных:
- `films.db` — каталог фильмов (таблицы `movies`, `genres`)
- `music.db` — каталог музыки (таблицы `Artist`, `Album`, `Track`, `Genre`)

## Запуск

### Docker (быстрый запуск с тестовыми данными)

Самый простой способ убедиться, что все задачи работают — запустить их в Docker.  
Это не требует установки Python и создаёт изолированное окружение с предустановленными тестовыми данными.

```bash
docker-compose up --build
```

Контейнер последовательно запустит все 15 задач с заранее подготовленными входными данными и выведет результаты в терминал.

### Локальный запуск (интерактивный режим)

Для более детальной проверки — запускайте скрипты вручную и вводите любые данные.  
Требования: **Python 3.x** (модуль `sqlite3` входит в стандартную библиотеку, ничего устанавливать не нужно).

**Задачи №1** — читают путь к файлу БД из стандартного ввода:

```bash
# Задача 1.1 — фильмы жанров 'музыка' и 'анимация' с 1997 года
python solution/task1/task1.py
# Введите путь: task/films.db

# Задача 1.5 — треки исполнителя (вводится путь к БД, затем имя артиста)
python solution/task1/task5.py
# Введите путь: task/music.db
# Введите имя: Led Zeppelin
```

**Задачи №2.1–2.8** — принимают путь к БД и вызывают `get_result(name)`:

```bash
python solution/task2/task1.py
# Введите путь: task/films.db
```

> Внимание: задачи №2.1–2.8 необратимо изменяют базу данных. Используйте копию файла.

**Задачи №2.9–2.10** — интерактивные программы с бесконечным циклом:

```bash
python solution/task2/task9.py  # Регистрация (Ctrl+C или Ctrl+D для выхода)
python solution/task2/task10.py # Авторизация (Ctrl+C или Ctrl+D для выхода)
```
